##Fract-olとは##

フラクタルとは複素数平面上で表される

ジュリア集合は上記の式を用いて

## Compiling and running
Run `make`.

Run it with `./fractol [fractal]` `j or m`. Possible fractals are: `mandelbrot`, `julia`,
Zoom in and out with the mousewheel. Only julia set you can change value of c. `./fractol [j] [real] [image]`
